## 快速使用

### 安装步骤：
- 安装好 docker && docker-compose 详情参考 docker 官方文档
- 打开终端，输入命令：`cd ~ && git clone https://github.com/smithyj/Docker.git`
- 输入命令：`echo "source ./Docker/.aliasrc" >> .bashrc && dc-start`

- 启动: `dc-start`
- 停止: `dc-stop`
- 重启: `dc-restart`
- 相关更多命令，请查看 `.aliasrc` 文件
